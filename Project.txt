План архитектуры

Обработчики
     Handler для управления постами:
        POST /posts: Создание нового поста.
        GET /posts: Получение списка всех постов.
        GET /posts/{postID}: Получение конкретного поста по его идентификатору.
        PUT /posts/{postID}: Обновление существующего поста.
        DELETE /posts/{postID}: Удаление поста.

    Handler для управления комментариями:
        POST /posts/{postID}/comments: Создание нового комментария к определенному посту.
        GET /posts/{postID}/comments: Получение списка всех комментариев к определенному посту.
        GET /posts/{postID}/comments/{commentID}: Получение конкретного комментария по его идентификатору.
        PUT /posts/{postID}/comments/{commentID}: Обновление существующего комментария.
        DELETE /posts/{postID}/comments/{commentID}: Удаление комментария.

    Handler для управления пользователями:
        POST /users/register: Регистрация нового пользователя.
        POST /users/login: Вход (аутентификация) пользователя.
        POST /users/logout: Выход (дезавторизация) пользователя.
        GET /users/me: Получение информации о текущем пользователе.
        PUT /users/me: Обновление информации о текущем пользователе.
        DELETE /users/me: Удаление учетной записи пользователя.
